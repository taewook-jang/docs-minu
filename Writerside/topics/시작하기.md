# 시작하기

## 개발 환경 설정하기
- 필수 프로그램 설치
    - JDK 17
    - IntelliJ IDEA 또는 Eclipse
    - Git
    - Gradle
- 프로젝트 설정
    - Git 저장소 클론
  ```bash
  git clone [repository-url]
  ```
    - Gradle 빌드
  ```bash
  ./gradlew clean build
  ```

## 프로젝트 구조 이해하기
- 주요 모듈
    - `_minu-code-gen`: 코드 생성 도구
    - `damda-system`: 시스템 관련 모듈
    - `damda-domain-data`: 도메인 데이터 모듈
    - `damda-api-client`: API 클라이언트 모듈
    - `damda-webapp`: 웹 애플리케이션 모듈
